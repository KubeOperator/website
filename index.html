什么是 FIT2OPENSHIFT ？

FIT2OPENSHIFT 是一个开源项目，帮助运维人员通过 Web 控制台，在完全离线环境下实现 OpenShift 社区版（okd）集群的可视化部署及运维管理。

  • 一键可视化部署
  • 提供各种部署模板，all-in-one模式、单master模式、多master模式等
  • 集群的弹性扩容、持续升级、备份及回滚
  • 成熟的网络、存储及安全方案
  • 提供 OpenShift 社区版各个版本离线包仓库，离线一键安装
Learn More

功能特点

可视化

安装过程可视化，不再只是全命令行操作，并可查看集群运行状况和部署架构。

简单易用

便捷的部署模板，只需几个配置项，就可以快速部署整套 OpenShift 环境。

离线安装

整个安装过程无需公网，解决了复杂网络下部署的难题。

多版本

提供 OpenShift 社区版各个版本离线包仓库，按需选择。

演示体验

快速开始

只需通过几行代码即可快速构建一个 OKD 环境
